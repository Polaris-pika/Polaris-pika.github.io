# vmware16 的下载与安装

我之前跟的那个教程给的和用的是 vmware15.5 的文件，这个版本强烈不推荐用，因为我用它的时候一旦我开启虚拟机的时候他就蓝屏（嘎嘎蓝），我感觉可能的问题是因为自己内核隔离没有关闭（我的还是正在运行）

不过我并没有去解决这个 bug，因为我升级到了 vmware16.0，然后他就解决了....nice！

然后这里附上 vmware 的下载链接，当然你要是害怕我分享的是流氓软件也可以去官网自己搜

# 虚拟机的安装

有很多虚拟机的版本你都是可以选择的，我这里安装的是一个 centos7.6 的老系统学习用（甚至都停服了），所以就只演示这个了，其他版本的系统方法也是类似的，百度找个镜像源下载一个就可以了。

VMware 左上角文件 -> 新建虚拟机 -> 先无脑下一步

这里你可以选择操作系统和版本，因为我这里的 centos 是 Red Hat 编译重构的一个社区开源版本所以直接选择这个版本就好了。

下一步让你修改虚拟机的名称和下载位置，有需要的记得改一下

下一步，需要你给你的虚拟机如果你磁盘充足的话。可以直接过，如果不是那么充足的话可以稍微压缩一点空间（扩展也没问题）选择将虚拟磁盘拆分成多个文件 -> 下一步。

现在就可以配置虚拟机的硬件内容了，根据自己的电脑实际情况和需要进行选择

# 选择 CD/DVD 路径

首先需要下载一下映像文件，你可以选择自己找镜像网下载（不过好像都下了），也可以下载我的链接

链接：https://pan.baidu.com/s/1hIRKOoj0m7W-6YcaGwaydw
提取码：mghv

点击你刚刚创建好的虚拟机，设备里面选择 CD/DVD（SATA）

选择你刚刚下载的文件的路径就好了

# 开启虚拟机

选择第一个 Install Centos7，语言选择中文（你也可以选择英文）

选择安装软件，默认是最小化安装，可以根据你自己的需要进行安装，你安装好之后有需要也可以重新安装，所以问题不大。这里我选择了 GNOME 桌面，并且勾选了两个兼容库和基本开发工具

# 安装位置，进行分区操作

其他存储选项 -> 我要配置分区

  1. 先指定 /boot 分区，即引导分区，大小为 1G, 然后点击添加挂载点。设备类型选择标准分区 -> 文件系统选择 ext4。
  2. 指定 swap 分区，大小为 2G 然后点击添加挂载点。设备类型选择标准分区 -> 文件系统选择 swap。
  3. 指定 / 分区，大小为剩下的容量点击添加挂载点。设备类型选择标准分区 -> 文件系统选择 ext4。

完成 -> 接受更改

KDUMP 根据需要自行选择勾不勾选。

# 设置网络和主机名，安装好后也可以设置

打开以太网的链接，起一个主机名，然后安装就可以了

# Root 密码和创建用户

设置 Root 密码（高权限），然后创建一个用户（默认用的是这个低权限的用户操作），也可以安装成功后，再创建。

重启一下，接受许可证（必须接受）

默认以普通用户登录就行，你也可以切换成 root, 点击未列出，用户名 root，输入密码

选择语言 -> 无脑前进

这里你可以连接你的在线账号，不连接你可以直接跳过

然后你就创建好了！
